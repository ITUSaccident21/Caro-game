## Cách Đọc Kết Quả Benchmark

Sau khi build và chạy game, console sẽ in:
```
[2026-05-23 10:30:15] HARD   d=6 move#12  | 143.2ms | 87,432 nodes |  5,201 pruned | 37.2% prune | root=28  | OK
```

Đọc theo thứ tự:
1. `HARD d=6` → chế độ và depth
2. `move#12` → nước thứ mấy (mid-game = nhiều candidates nhất)
3. `143.2ms` → thời gian — so với ngưỡng HARD: OK (<500ms), Good (<200ms), Excellent (<80ms)
4. `37.2% prune` → hiệu quả Alpha-Beta — thấp → cần move ordering
