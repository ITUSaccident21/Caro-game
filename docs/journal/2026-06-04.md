## Style Analysis

Xem xét 5 screenshot Stardew Valley + 2 tham chiếu trước đó.
Kết luận: style gọi là "**Cozy Pixel Art**" — 16-bit, warm earth tones, vật liệu gỗ/parchment.

Đặc trưng nhận diện:
- Background: sky gradient hoặc solid màu pastel + bụi cây góc dưới
- Panel: warm beige (#D4B486) với viền gỗ nâu (#6B3818)
- Button: parchment fill, khi hover/select → gold highlight
- Font: m5x7 (pixel serif mềm, không blocky như Press Start 2P)

## Kỹ Thuật Mới Học

### `IMG_LoadTexture(renderer, path)`
Load PNG/JPG trực tiếp thành GPU texture trong 1 dòng.
Không cần Surface intermediate (khác `SDL_CreateTextureFromSurface`).
Return nullptr nếu file không tồn tại — cần check trước khi dùng.

### Graceful Degradation
Mọi texture đều có fallback:
- Không có board.jpg → vẽ checkerboard procedural
- Không có bg_menu.png → vẽ solid color
- Không có char sprites → vẽ colored square + chữ X/O

Đây là nguyên tắc quan trọng: game vẫn chạy được dù thiếu asset.

### Sprite Layer Compositing
```cpp
for (int i = 0; i < 5; i++)
    SDL_RenderCopy(r, layers[i], &src, &dst);  // cùng dst rect
```
SDL2 alpha blending tự động blend layer trên lên layer dưới.
Thứ tự: base → top → bottom → hair → eyes
→ Tóc và mắt nằm trên cùng, cơ thể ở dưới.

### `SDL_QueryTexture`
```cpp
int tw, th;
SDL_QueryTexture(tex, nullptr, nullptr, &tw, &th);
// Sprite sheet 2×3: mỗi frame = (tw/2, th/3)
SDL_Rect src = {0, 0, tw/2, th/3};  // frame đầu tiên
```
Tính kích thước frame động thay vì hardcode.
